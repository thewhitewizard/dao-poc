 # NOP-001 · Fund Open-Source Confidential Governance SDK

  ## Summary

  Allocate **50 000 TWW** from the NoxGov treasury to fund a 3-month open-source
  sprint delivering a public **Confidential Governance SDK** — so any DAO can
  integrate Nox Protocol-based private voting in under a day.

  ## Motivation

  NoxGov is the first production DAO to run fully encrypted votes (choice + weight)
  on-chain. The infrastructure we built — `encryptInput`, `publicDecrypt`, vote
  receipts, liquid delegation — is reusable but not yet packaged for external teams.

  A public SDK would:
  - Accelerate adoption of confidential governance across the ecosystem
  - Position TWW as the reference staking asset for privacy-preserving DAOs
  - Generate protocol fees through SDK integrations

  ## Specification

  ### Deliverables (3 months)

  | # | Deliverable | Owner |
  |---|-------------|-------|
  | 1 | `@noxgov/sdk` npm package with React hooks | Core team |
  | 2 | Documentation site (docs.noxgov.xyz) | Core team |
  | 3 | 2 reference integrations (Snapshot fork + custom DAO) | Community |
  | 4 | Security audit of the delegation module | External |

  June 2026    → Audit + public launch
